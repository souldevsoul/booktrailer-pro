## RECOMMENDED TOOLS

1. **Recraft V3** - Best for logos, icons, vector graphics
2. **FLUX Pro** - Best for photorealistic images, UI mockups, portraits
3. **Midjourney** - Alternative for all image types
4. **Figma/Canva** - For text overlays and final compositions

## EXPORT SETTINGS

- **Logos/Icons:** SVG or PNG with transparency, 2x resolution for retina
- **Photos/Mockups:** JPG or WebP, optimized for web (quality 85%)
- **Hero images:** WebP format, progressive loading, max 500KB
- **Thumbnails:** WebP format, max 200KB

## COLOR PALETTE REFERENCE

Use these exact colors for consistency:
